# Nobel Prize Data Analysis

This project explores Nobel Prize winners with a focus on:
- Gender representation
- Category distribution
- Historical trends
- Female participation over time

## Key Questions
- Which categories dominate Nobel Prizes?
- Has female participation increased over decades?
- When did women first enter each category?
- What is the gender gap across fields?

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Female participation has increased steadily since the 1970s.
- Literature and Peace were among the earliest categories with female laureates.
- Some scientific fields show long gender gaps.
